Unit I: Errors in Analysis

Types of Errors

Errors affect the accuracy and precision of results.

Determinate (Systematic)

Avoidable errors with identifiable causes (e.g., faulty instrument, personal bias).

Indeterminate (Random)

Unavoidable errors due to random fluctuations (e.g., temperature changes).

Mnemonic for Methods to Minimize Errors: "C-R-A-B"
C = Calibration, R = Running blank, A = Average (Mean), B = Blinding
